A small status or count pill; the `tone` maps to a semantic role color.

```jsx
<Badge tone="error" icon="unlink">3 broken</Badge>
<Badge tone="saved" dot>Saved Layout</Badge>
<Badge tone="generated" dot>Generated Layout</Badge>
<Badge tone="dirty" dot>Unsaved</Badge>
<Badge tone="neutral" mono>v3.3.7</Badge>
```

Tones: `neutral · link · tag · var · warn · error · dirty · saved · generated · build`. Use `dot` for a status indicator, `icon` for a Tabler glyph, `mono` for versions/IDs.
